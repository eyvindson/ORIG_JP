Projektin rakenne:

* Kaikki syöttödataa menee kansioon Input
* Kaikki välitulokset menevät kansioon Midresults
* Kaikki lopulliset tulokset kansioon Results
* Mitään polkuja ei ole kirjoitettu suoraan koodiin

Käyttöohje:

1. avaa tiedosto PATHS.R ja ohjaa polku siihen kansioon jossa laskutiedostot sijaitsevat
2. varmista, että kaikki tarvittavat paketit on asennettu (LIBRARIES.R)
3. aja main.R
4. tulokset menevät kansioon results, raakakuvat Figures ja julkaisukuvat sen alakansioon Publication
